# Data Science Job Market & Salary Analysis

An exploratory analysis of data science job salaries — what roles, experience levels, company sizes, and work arrangements drive compensation.

## 🎯 Project Goal

Understand the entry-level data science job market and build a model to predict salary based on role, experience, and location.

## 📊 Key Questions

1. What roles pay the most? (DS vs. MLE vs. DA vs. DE)
2. How does experience level affect salary?
3. Remote vs. on-site — does it matter for pay?
4. Which company sizes pay better?
5. Can we predict salary from role + experience + location?

## 🛠 Tech Stack

- Python (pandas, NumPy, scikit-learn, matplotlib, seaborn)
- Jupyter Notebook
